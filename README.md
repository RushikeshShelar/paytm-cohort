
## Build a basic version of PayTM
An App built during Cohort 2.0 by HArkirat Singh

Tech Used:
Frontend: React.js
Backend : Node.js , Express.js
Database : MongoDB


Features:
- View Balance
- Find All Users 
- Send Money
- Receive Money
- Sign In
- Sign Up

Api End Points:
Users
- POST /users/signin
- POST /users/signup
- PUT /users
- GET /users/bulk?filter=

Account
- GET /balance
- POST /transfer
